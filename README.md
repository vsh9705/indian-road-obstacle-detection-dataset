# Indian Road Obstacle Detection Dataset

This repository contains a curated object detection dataset focused on identifying common road obstacles found in Indian road environments.

## Classes

The dataset contains annotations for the following classes:

0 pothole  
1 thela  
2 animal  
3 barricade  
4 rickshaw  

## Dataset Structure

images/
annotations/
metadata/
docs/

## Annotation Format

Annotations follow YOLO format:

class_id x_center y_center width height

All coordinates are normalized between 0 and 1.

## Applications

Autonomous driving research  
Road safety analysis  
Traffic monitoring  
Obstacle detection systems  

## Dataset Highlights

Focus on obstacles unique to Indian road conditions.  
Includes diverse urban traffic environments.  
Manually annotated bounding boxes.
