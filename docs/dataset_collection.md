# Dataset Collection

## Overview

The Indian Road Obstacle Detection Dataset was created to support research in object detection for road safety and autonomous driving systems in Indian traffic environments.

Indian roads often contain obstacles that are uncommon in many existing driving datasets. This dataset focuses on detecting such obstacles to improve real world applicability of computer vision models.

## Target Classes

The dataset contains five object categories that frequently appear on Indian roads:

1. pothole
2. thela (street vendor cart)
3. animal
4. barricade
5. rickshaw

These categories were selected because they represent common obstacles that affect road safety and vehicle navigation.

## Image Sources

Images were collected from publicly available road scene photographs and manually curated to represent Indian urban and semi urban environments.

The dataset includes images captured in different conditions such as:

* daylight and evening lighting
* varying traffic density
* different camera angles
* partial occlusion of objects
* varying object sizes and distances

## Dataset Structure

The dataset is organized into three subsets:

* training set
* validation set
* testing set

This split allows models to be trained, tuned, and evaluated properly.

## Applications

This dataset can be used for:

* object detection research
* autonomous driving systems
* traffic monitoring systems
* road safety analysis
* obstacle detection in navigation systems
